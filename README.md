# Incredible Community Marketplace

Custom WordPress marketplace plugin (JQ Marketplace Engine) powering a gated marketplace for three listing types: equipment rentals, equipment sales, and service bookings. Built for joequick.com.

The platform (Warrior / TCP) acts as a marketplace facilitator only — not a guarantor, insurer, or legal judge.

## Transaction Types

1. **Equipment Rental** — Warrior equipment listed by approved TCP members
2. **Equipment Sale** — Warrior equipment or related tool sales
3. **Service Booking** — Certified Coatings Consultants list and get booked locally

## Tech Stack

- WordPress + WooCommerce (account/checkout foundation)
- Custom plugin: `jq-marketplace-engine`
- Custom database tables for transactional/financial data
- Policy engine with admin-configurable business rules
